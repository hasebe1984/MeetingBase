# 社内会議室予約システム (Meeting Room Reservation System)

職業訓練校のチーム開発課題として制作した、Web会議室予約システムです。
「紙やExcelでの管理によるダブルブッキングの解消」を目的としています。

## 💻 担当業務と実績

### 1. プロジェクトマネジメント（PL）
**「遅れを放置しない進行管理」**
- **予実管理の徹底:**
  WBSを用いて日次で進捗を確認。開発中盤で発生した遅れに対し、技術リーダーと連携してタスクを再配分し、納期内での納品を実現しました。
- **品質管理:**
  不具合管理表を導入し、バグの発見・修正状況を可視化。約20件のクリティカルな不具合を解消しました。

### 2. 機能実装（バックエンド）
システムの中核となる「会員管理」および「管理者機能」の全般を担当しました。

**【担当機能詳細】**
- **会員機能:**
  - 新規登録・編集・退会処理 (`RegistrationServlet`, `UserEditServlet`)
  - バリデーションチェック（入力値の型・桁数・必須チェック）
- **管理者機能:**
  - ユーザー一覧表示・権限変更 (`AdminUserServlet`)
  - 会議室マスタ管理（追加・変更・削除）
- **セキュリティ:**
  - 権限チェックフィルターの実装

### 3. 問題解決のエピソード
**課題：管理者が不在になる不具合の解消**
テスト工程にて、「管理者が自分自身の権限を『一般』に変更できてしまい、システム上に管理者が一人もいなくなる」という重大な欠陥（不具合管理ID:36）が発覚しました。

**解決策：論理チェックの導入**
「操作しているユーザー自身が管理者の場合、自身の権限変更（降格）は不可とする」というロジックを実装し、システムの整合性を保つよう修正しました。
