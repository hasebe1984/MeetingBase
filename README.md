# 社内会議室予約システム (Meeting Room Reservation System)

職業訓練校のチーム開発課題として制作した、Web会議室予約システムです。
「紙やExcelでの管理によるダブルブッキングの解消」を目的としています。

## 開発環境・技術
- **言語:** Java 17 (Servlet / JSP), HTML, CSS
- **DB:** MySQL 8.0
- **ツール:** Eclipse, Tomcat 9, Git/GitHub, Docker
- **開発期間:** 2026年1月6日 ～ 2026年2月3日（約1ヶ月）

## チーム体制
- **人数:** 7名
- **私の役割:** プロジェクトリーダー (PL) 兼 開発担当

## 担当業務と成果

### 1. プロジェクトマネジメント（PL）
**「遅れを放置しない進行管理」**
- **WBSによる進捗管理:** 日次で予実管理を実施。遅れが発生した際は、技術リーダーと連携して開発リソースを再配分し、納期遅延を防ぎました。
- **品質管理:**
  不具合管理表を導入し、バグの発見・修正状況を可視化。約20件の不具合を解消し、品質を担保しました。

### 2. 機能実装（バックエンド）
**「管理者・会員機能の実装」**
システムの中核となる会員管理機能（登録・編集・退会）および、管理者専用機能の実装を担当しました。

**【工夫した点：論理的な整合性の確保】**
テスト段階で「管理者が自身の権限を一般ユーザーに変更できてしまい、システムから管理者が不在になる」という不具合（ID:36）が発覚。
これに対し、「操作中のユーザーが管理者の場合、自身の権限変更を不可にする」というロジックを追加し、システムの破綻を防ぎました。

## 機能一覧
| カテゴリ | 機能詳細 |
| --- | --- |
| **認証** | ログイン、ログアウト |
| **予約** | 会議室の空き状況検索、予約登録、キャンセル |
| **会員** | 新規登録、会員情報編集、退会 |
| **管理** | 会員一覧表示、権限変更、会議室の追加・削除 |
